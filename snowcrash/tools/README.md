## Snowcrash Helper Tools

This directory contains small helper programs used while working on Snowcrash levels.

- `caesar.c`: quick Caesar-shift brute-force helper used for simple substitution checks.
- `show_ids.c`: prints real and effective user IDs (`RUID`/`EUID`) for privilege-context verification.

Only Snowcrash-relevant helpers are kept here to avoid unrelated noise in the portfolio.
